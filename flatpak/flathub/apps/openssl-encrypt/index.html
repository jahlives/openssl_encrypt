📖 Description

OpenSSL Encrypt is a comprehensive encryption tool that provides military-grade security with cutting-edge post-quantum cryptography. Protect your files against both current and future quantum computing threats.

Perfect for individuals and organizations requiring the highest levels of data protection, including protection against future quantum computer attacks.

★ Features

🔐 Encryption Algorithms

  • AES-GCM (128/256-bit)
  • ChaCha20-Poly1305
  • XChaCha20-Poly1305
  • Fernet (AES-128-CBC)
  • AES-GCM-SIV

⚛ Post-Quantum Cryptography

  • ML-KEM (512/768/1024-bit)
  • HQC (128/192/256-bit)
  • MAYO (1/3/5 multivariate)
  • CROSS (128/192/256 code-based)
  • Kyber hybrid encryption
  • Quantum-resistant key exchange
  • Future-proof security

🔑 Key Derivation Functions

  • HKDF (RFC 5869 modern standard)
  • Argon2 (recommended)
  • Scrypt
  • Balloon Hashing
  • PBKDF2 (legacy)
  • Configurable iterations
  • Salt generation

📝 Hash Algorithms

  • SHA-2 Family (224/256/384/512-bit)
  • SHA-3 Family (224/256/384/512-bit)
  • BLAKE3 (ultra-fast tree-based)
  • BLAKE2b (high-performance)
  • SHAKE (128/256 extendable-output)
  • Whirlpool (legacy)

💻 User Interface

  • Modern GUI with dark/light themes
  • Command-line interface
  • Comprehensive debug mode with --debug flag
  • Drag & drop file support
  • Progress indicators
  • Batch processing

🔒 Security Features

  • Secure memory handling
  • Military-grade secure deletion
  • Constant-time operations
  • Password strength validation
  • Environment variable support
